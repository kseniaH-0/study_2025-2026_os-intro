# Контрольные вопросы

1. Какие режимы работы есть в mc? Охарактеризуйте их.

Две панели (основной режим), одна панель, информация, дерево, быстрый просмотр.

2. Какие операции с файлами можно выполнить как с помощью команд shell, так и с помощью меню mc? Приведите примеры.

Копирование (cp и F5), перемещение (mv и F6), удаление (rm и F8), создание каталога (mkdir и F7).


3. Опишите структуру меню левой (или правой) панели mc, дайте характеристику командам.

List mode (режим списка), Quick view (быстрый просмотр), Info (информация о каталоге), Tree (дерево каталогов).

4. Опишите структуру меню Файл mc, дайте характеристику командам.

Просмотр (F3), правка (F4), копирование (F5), перемещение (F6), создание каталога (F7), удаление (F8), выход (F10).

5. Опишите структуру меню Команда mc, дайте характеристику командам.

Дерево каталогов, поиск файла, сравнение каталогов, история команд, каталоги быстрого доступа, восстановление файлов.

6. Опишите структуру меню Настройки mc, дайте характеристику командам.

Конфигурация панелей, внешний вид, подтверждения, распознавание клавиш, виртуальная ФС.

7. Назовите и дайте характеристику встроенным командам mc.

F1 — помощь, F2 — пользовательское меню, F3 — просмотр, F4 — правка, F5 — копирование, F6 — перемещение, F7 — создание каталога, F8 — удаление, F9 — меню, F10 — выход.

8. Назовите и дайте характеристику командам встроенного редактора mc.

Ctrl+Y — удалить строку, F5 — копировать выделенное, F6 — вырезать/вставить, Ctrl+U — отменить, F2 — сохранить, Ctrl+Home/End — начало/конец файла.

9. Дайте характеристику средствам mc, которые позволяют создавать меню, определяемые пользователем.

Файл ~/.mc/menu позволяет создавать пользовательские пункты меню.

10. Дайте характеристику средствам mc, которые позволяют выполнять действия, определяемые пользователем, над текущим файлом.

Файл ~/.mc/ext позволяет задавать действия для файлов с определёнными расширениями.
